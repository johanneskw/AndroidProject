Gruppeprosjekt i IN2000 - Software engineering og prosjektarbeid. Prosjektet gikk ut på å lage en Android app i Android Studios med data fra ett midlertidig API fra Meterologisk institutt. 
Appens funksjon er å hente data om luftkvaliteten i byer i Norge, og fremstille data på en forståelig funksjonell måte. Appen viser luftkvaliteten i byer på ett kart fragment, hvor man kan trykke på markører for mer informasjon, eller legge til by i en favorittliste. 
Informasjon om luftkvalitet på de forskjellige målestasjonene kan leses i ett liste-view, med interaktive informasjon-kort om hver stasjon. I liste-viewet kan man legge til sine "favoritt" stasjoner, samt se informasjon om sin nærmeste målestasjon. 
Ved å klikke på det interaktive informasjons-kortet vil mer utdypende data dukke opp, og man kan navigere seg gjennom en meny for å se data om mengder av partikler og støv i luften fremstilt i grafer og skjema. 

Appen kjører på Android 7.0, API 24 eller senere.
